Web Components

For Creating Websites Rapidly!

Why Rapid Build UI?

  • Components are html elements. So if you know html then you're all set! Some components require basic javascript knowledge. Ease of Use
  • Components are styled and function to provide the best user experience. User Friendly
  • Components are built to look and function appropriately on desktops and mobile devices. Mobile Friendly
  • Just npm install, javascript import then place the html tag anywhere! See the getting started. Simple Installation
  • All styles are backed in. No need to worry about writing css. Woohoo! No Styling Required
  • Css variables are available for every component to provide your own style. Customizable Styles
  • No javascript framework required. Components are native to the browser. Meaning, they will work with or without a javascript framework (like angular or react). Framework Agnostic
  • Every component is its own package. So if you only want to use one, no problemo! Use All or Individually
  • Documentation is just as important to us as creating the components. This site contains interactive demos for every component to see exactly what each feature does. Superb Documentation
  • Refer to the browser support page. Microsoft Edge is currently unsupported. However, support is coming soon... Excellent Browser Support
