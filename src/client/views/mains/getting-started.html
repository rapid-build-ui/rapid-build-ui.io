Getting Started

Status: In Development

The components are being worked on just about everyday. No official releases have been cut yet.

Prerequisites

  • Node.js
  • npm or yarn
    • Only one version of a component can be installed.
    • If you use npm instead of yarn make sure the components are installed flat.
    • Yarn has a built in feature: yarn install --flat

How to Use

Every component is its own package.
So if you only want to use one, no problemo!

  1. Click the install icon on any of the component pages and run the command.
    Example:
  2. Once installed, components can be loaded via 2 ways choose one.
    Example (type="module" required on script tag): <!doctype html> <html> <head> <!-- Load via script tag: --> <script type="module" src="/node_modules/@rapid-build-ui/rb-alert/scripts/rb-alert.js"> </script> <!-- OR --> <!-- Load via javascript import: --> <script type="module"> import '/node_modules/@rapid-build-ui/rb-alert/scripts/rb-alert.js'; </script> </head> <body> <!-- Use rb-component anywhere! --> <rb-alert>Hello Gorgeous!</rb-alert> </body> </html>
