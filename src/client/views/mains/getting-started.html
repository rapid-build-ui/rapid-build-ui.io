Getting Started

Status: In Development

The components are in early stages of development.

Prerequisites (these must be installed first)

  • Node.js
  • Yarn
    • You can use npm instead just make sure the components are installed flat.
    • Only one version of each component can be installed.
    • Yarn has a built in feature: yarn install --flat.

How to Use (each component is it's own package)
