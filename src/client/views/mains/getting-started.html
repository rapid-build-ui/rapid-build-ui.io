Getting Started

Status: In Development

The components are in early stages of development.

Prerequisites (these must be installed first)

  • Node.js
  • Yarn
    • You can use npm instead, just make sure the components are installed flat.
    • Only one version of each component can be installed.
    • Yarn has a built in feature: yarn install --flat.

How to Use (each component is it's own package)

  1. Click on the install icon on any of the component pages.
    • Example:
  2. Once installed place the component script tag in your html.
    • Example: <script type="module" src="/node_modules/@rapid-build-ui/rb-alert/scripts/rb-alert.js"> </script>
  3. The component is ready! Use it anywhere in your html.
    • Example: <rb-alert> Hello Gorgeous! </rb-alert>
